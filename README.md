# JianLaiRAG

一个基于检索增强生成（RAG）技术的智能问答系统。

## 项目简介

JianLaiRAG 是一个检索增强生成（Retrieval-Augmented Generation）项目，旨在通过结合信息检索和生成式 AI 技术，提供更准确、更可靠的智能问答能力。

## 功能特性

- 🔍 **智能检索**：高效的文档检索和语义搜索
- 🤖 **生成式问答**：基于检索内容生成准确答案
- 📚 **多文档支持**：支持多种文档格式和来源
- 🚀 **高性能**：优化的检索和生成流程
